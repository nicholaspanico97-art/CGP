# CGP — Capital Generation Program

Standalone project, separate from the workshop repo.

Purpose: identify, evaluate, and execute ways to generate capital.
Scope, targets, and first workstreams TBD — see `PROJECT_STATE.md`,
which is the living source of truth for this project.

## Repo layout
- `PROJECT_STATE.md` — goals, ledger, decision log, next actions
- `workstreams/` — one folder per active income/capital workstream
- `research/` — evaluation notes on candidate ideas (kept even if rejected)
